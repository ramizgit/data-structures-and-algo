1. 0/1 Knapsack Family (must know) : These all use a backward loop because each item can be used at most once.

| Problem                       | Pattern                             |
| ----------------------------- | ----------------------------------- |
| Partition Equal Subset Sum    | Boolean DP                          |
| Last Stone Weight II          | Boolean DP                          |
| Ones and Zeroes               | 2D Knapsack                         |
| Target Sum                    | Count DP (or subset transformation) |
| Count of Subsets With Sum K   | Count DP                            |
| Minimum Subset Sum Difference | Boolean DP                          |


2. Unbounded Knapsack Family (must know) : These use a forward loop because each item can be reused.
| Problem            | Pattern                 |
| ------------------ | ----------------------- |
| Coin Change        | Min DP                  |
| Coin Change II     | Count DP                |
| Perfect Squares    | Min DP                  |
| Combination Sum IV | Count DP (permutations) |
| Integer Break      | Max DP                  |
| Rod Cutting        | Max DP                  |
