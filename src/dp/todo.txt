1. 0/1 Knapsack Family (must know) : These all use a backward loop because each item can be used at most once.

| Problem                       | Pattern                             |
| ----------------------------- | ----------------------------------- |
| Partition Equal Subset Sum    | Boolean DP                          | [Done]
| Last Stone Weight II          | Boolean DP                          | [Done]
| Ones and Zeroes               | 2D Knapsack                         |
| Target Sum                    | Count DP (or subset transformation) | [Done]
| Count of Subsets With Sum K   | Count DP                            | [Done]
| Minimum Subset Sum Difference | Boolean DP                          | [Done]
